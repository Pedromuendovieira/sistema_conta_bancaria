# Sistema Bancário em Java

Projeto desenvolvido em Java com foco em Programação Orientada a Objetos (POO).

## Funcionalidades

- Criar conta bancária
- Depositar valores
- Levantar valores
- Consultar saldo
- Testes unitários com JUnit

## Tecnologias Utilizadas

- Java
- IntelliJ IDEA
- Git
- GitHub
- JUnit 5
